## Installation
To install my project 

clone repository

```bash
git clone https://github.com/Wasayshaikh/FarmByte-assessment.git

```
Open project directory

```bash
cd FarmByte-assessment

```

Install dependencies

```bash
npm Install

```
Run project

```bash
npm run dev

```
